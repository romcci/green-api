# Green API Integration Panel

## Описание проекта

HTML-страница для интеграции с WhatsApp через Green API с визуальным интерфейсом для тестирования основных методов API. **Страница разделена на две половины: левая часть содержит формы ввода, правая часть отображает ответы API.**

## Основные возможности

✅ **Реализованные методы:**
- `getSettings` - Получение настроек инстанса
- `getStateInstance` - Получение состояния инстанса  
- `sendMessage` - Отправка текстовых сообщений
- `sendFileByUrl` - Отправка файлов по URL

✅ **Функциональность:**
- **Раздельный экран: левая часть для ввода, правая часть для ответов**
- Поля ввода для `idInstance` и `ApiTokenInstance`
- Отдельные поля для параметров сообщений и файлов
- **Область отображения ответов API на всю высоту экрана (только для чтения)**
- Тёмная тема с зелёными акцентами
- Адаптивный интерфейс для различных устройств
- Обработка ошибок и валидация входных данных

## Как использовать

### 1. Получение параметров подключения
1. Перейдите в личный кабинет Green API
2. Создайте новый инстанс на бесплатном аккаунте разработчика
3. Сканируйте QR-код для подключения своего номера телефона
4. Скопируйте `idInstance` и `ApiTokenInstance` из личного кабинета

### 2. Использование страницы
1. Откройте страницу в веб-браузере
2. Введите `idInstance` и `ApiTokenInstance` в соответствующие поля **(левая часть экрана)**
3. Заполните дополнительные поля для номеров телефонов и сообщений при необходимости
4. Нажимайте кнопки методов последовательно **(левая часть экрана)**:
   - `Get Settings` - для получения настроек
   - `Get State Instance` - для получения состояния инстанса
   - `Send Message` - для отправки сообщений
   - `Send File by URL` - для отправки файлов
5. **Наблюдайте результаты работы методов в правой части экрана**

### 3. Параметры методов

#### sendMessage:
- Номер телефона: в формате `79001234567` (без + и пробелов)
- Текст сообщения: любой текст

#### sendFileByUrl:
- Номер телефона: в формате `79001234567`
- URL файла: прямая ссылка на файл (jpg, png, pdf, и т.д.)

## Технические детали

### Новый макет страницы
- **Левая половина экрана**: все элементы управления и поля ввода в строгом порядке
- **Правая половина экрана**: область ответов API на всю высоту экрана
- **Тёмная цветовая схема**: оптимизирована для длительной работы

### Структура проекта
```
index.html - Основная HTML-страница с интерфейсом
```

### Используемые технологии
- HTML5
- CSS3 (Tailwind CSS через CDN)
- JavaScript (ES6+)
- Font Awesome для иконок

### API Endpoints
```
GET  https://api.green-api.com/waInstance{idInstance}/getSettings/{apiTokenInstance}
GET  https://api.green-api.com/waInstance{idInstance}/getStateInstance/{apiTokenInstance}
POST https://api.green-api.com/waInstance{idInstance}/sendMessage/{apiTokenInstance}
POST https://api.green-api.com/waInstance{idInstance}/sendFileByUrl/{apiTokenInstance}
```

### Особенности реализации
- Все запросы выполняются с клиентской стороны через fetch API
- Обработка CORS реализована на стороне Green API
- Форматирование ответов в JSON для удобства чтения
- Автоматическое добавление суффикса '@c.us' к номерам телефонов

## Требования

- Современный веб-браузер с поддержкой ES6+
- Активное интернет-соединение
- Действующий аккаунт Green API с бесплатным тарифом

## Безопасность

- Все параметры подключения хранятся только в браузере
- Нет серверной части - все запросы выполняются клиентом
- Используйте только надежные сети для тестирования
